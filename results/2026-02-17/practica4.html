Ordenar listas, el codigo usado ordena el objeto que ya existe, no crea otro

order(help) genomas <- genomas[order(genomas$Assembly.Release.Date), ] kable(genomas) %>% kable_styling() # Ordenado por el año de lanzamiento del ensamblamiento, en ascendente

genomas\(Assembly.Release.Date <- as.Date(genomas\)Assembly.Release.Date, ‘%Y-%m-%d’) # Esto permite atribuir a la columna de fecha de lanzamiento un valor de # fecha, que no se lea como texto, sino como numero

Aquí usamos “cumsum()” para obtener la suma acumulada del número de bases

secuenciadas.

plot(x = genomas\(Assembly.Release.Date, y = cumsum(genomas\)Assembly.Stats.ATGC.Count), type = ‘b’, xlab = ‘Fecha de publicación’, ylab = ‘Longitud total secuenciada (bp)’, main = ‘Genomas anotados de Trypanosoma’) # Ahora que la columna de la que hablabamos se toma como una fecha podemos # crear una tabla que saque por pantalla el numero de secuencias lanzadas # en cada año
